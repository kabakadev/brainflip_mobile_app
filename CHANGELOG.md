# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-13

### Added
- Initial BrainFlip MVP for Android
- Email/password auth and Google sign-in
- Spaced repetition study sessions
- Prebuilt STEM decks and onboarding deck selection
- Community decks: browse and view shared decks
- Basic gamification: streaks, daily goals, session celebrations
- Progress tracking (study stats, completion screens)
- Custom app icon and splash screen
